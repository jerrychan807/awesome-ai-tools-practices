# 需求:

网页内容太多，想要总结出概览，或者快速找到想要的东西，一般我们只是在网页上搜索关键词，但是这样没办法进行模糊搜索。
需求就是我只需要大概描述我想要的东西，让 ai 能帮忙在整个页面找出想要的部分。

# 初始思路:

Arc browser can do that!
